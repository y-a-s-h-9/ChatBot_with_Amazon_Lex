# ChatBot_with_Amazon_Lex
A chatbot built with Amazon Lex for creating conversational interfaces using natural language understanding.

### Step #1
#### Create a Lex Bot
- Sign in to the AWS Management Console and navigate to Amazon Lex.

- Click Create Bot and choose a bot template or build from scratch.

- Provide a Bot name, language, and IAM role for permissions.

#### Define Intents
- Create intents, which represent the user's goals.

- Add sample utterances (phrases users might say to trigger the intent).

- Configure a confirmation prompt if necessary to verify user intent.
